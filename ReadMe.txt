# P6_JustStreamIt
Développez une interface utilisateur pour une application web Python


Après avoir contacté Camille la CEO de JustStreamIt pour le projet de developpement d'un interface utilisateur et eu les informations de la part de Eric sur la compatibilité souhaité des navigateurs web les plus utilisés et leurs API " OCMovies-API ".

A l'aide du dépot de code reçu par Eric contenants les étapes permettant de lancer des requêtes à l'API j'ai procédé aux étapes suivantes :

1 - Construction de la maquette en html.
2 - Utilisation de javascript natif.
3 - Consommation de l'API & utilisation des filtres mentionnés dans le readMe pour la récupération des données.
4 - Utilisation de W3C pour résoudre les différentes erreurs possible durant le développement.
5 - Gestion du slider au click afin d'afficher les 7 films demandé par categories.
6 - Construction et affichage de la modal au click en html & javascript.
7 - Design de la maquette en css.
